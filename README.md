# HouseMates

**Smart shared living for housemates.**

HouseMates replaces the messy combination of WhatsApp groups, spreadsheets, and Splitwise
with one clean app built specifically for people sharing a home.

---

## What It Does

- **Bills** — Track who paid what, see balances at a glance, get reminded before due dates
- **Parking** — See who has the spot, reserve it in advance, approve requests
- **Groceries** — Shared lists + personal lists, real-time sync
- **Chores** — Assign, rotate, track completion
- **Tasks** — House to-dos, notice board, shared notes
- **Chat** — House group chat + shared photo album
